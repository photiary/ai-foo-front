# 페이지: 식사 이미지 분석 업로드 화면

- 도메인: apps/web/app/food
- URL: /food/analysis

- cachedTokens '입력 토큰(Cached)' 항목 추가하여 표시
- cachedInputCost '입력 비용(Cached)' 항목 추가하여 표시
